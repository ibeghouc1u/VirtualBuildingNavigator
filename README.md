# 🏠 Virtual Building Navigator

An Android application that allows users to create, save, and explore virtual representations of real-world indoor spaces using photographs captured from the four cardinal directions (North, South, East, and West).

The project aims to provide a lightweight virtual tour system where users can model buildings, define room connections, and navigate interactively between rooms through clickable access points.

---

## 📖 Overview

Virtual Building Navigator enables users to digitally recreate indoor environments such as:

- Houses
- Apartments
- Offices
- Educational buildings
- Commercial spaces

Instead of relying on complex 3D modeling techniques, the application uses directional photographs combined with room connectivity information to create an intuitive navigation experience.

---

## ✨ Features

### 🏗️ Building Modeling

- Create a new building project.
- Add multiple rooms.
- Assign a unique name to each room.
- Capture photographs for:
  - North wall
  - South wall
  - East wall
  - West wall

### 🚪 Room Connections

- Define doors and passageways between rooms.
- Create interactive access points directly on wall images.
- Link rooms together to build a complete navigation graph.

### 🧭 Virtual Navigation

- Select a starting room.
- Choose an initial orientation.
- Rotate left and right to view different walls.
- Click on access points to move between connected rooms.
- Explore the entire building interactively.

### 💾 Data Persistence

- Save projects locally.
- Load previously created building models.
- Store all project information using JSON serialization.

---

## 📱 Application Workflow

### Model Creation

1. Create a new project.
2. Enter a building name.
3. Add a room.
4. Capture photos of the four walls.
5. Define room connections.
6. Save the project.

### Model Exploration

1. Load an existing project.
2. Choose a starting room.
3. Select an orientation.
4. Navigate through the virtual environment.
5. Move between connected rooms using interactive access points.
